# simple-app-cue

This directory contains the CUE output generated by helm2cue from the
[`simple-app`](../simple-app/) Helm chart. It is produced automatically via
`go generate` (see [`gen.go`](../../gen.go)) and committed so that the
conversion result is browsable without running the tool.

## Files

| File | Description |
|------|-------------|
| `values.yaml` | Default values copied from the source chart. |
| `values.cue` | CUE schema for the values, with defaults. |
| `context.cue` | Chart metadata and release context (`#chart`, `#release`). |
| `data.cue` | Wiring: embeds `values.yaml` and `release.yaml` into the CUE values. |
| `helpers.cue` | Converted `_helpers.tpl`: name, fullname, labels, etc. |
| `configmap.cue` | Converted `templates/configmap.yaml`. |
| `deployment.cue` | Converted `templates/deployment.yaml`. |
| `service.cue` | Converted `templates/service.yaml`. |
| `results.cue` | Aggregates all resource outputs into a single list. |
| `release.yaml` | Release-time values (populated at evaluation time). |
| `cue.mod/` | CUE module definition. |

## Evaluating

Export the rendered manifests as YAML:

```bash
cue export ./examples/simple-app-cue -t release_name=my-release --out yaml -e results
```

## Regenerating

```bash
go generate ./...
```
